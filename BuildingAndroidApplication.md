# Introduction #

Android development is very popular nowadays due to huge numbers of android device users. Android is not too old technology but has many versions with a lot of differences  from 2008 to 2015 which are 1.0 to 5.0. To develop an app wee need special android development environments such as eclipse with android SDK. An other point of android development is that testing which differs from desktop or web development.
